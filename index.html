What is ripplePay?

ripplePay is a mobile app that seamlessly allows users to deposit, send, & request Ripple XRP. By integrating the Ripple Labs API, ripplePay gives users the ability to send funds anywhere in the world in a matter of seconds - absolutely free.

Technologies

ripplePay utilizes the most recent and up to date technologies to deliver a secure and performing experience.


    MongoDB

    React Native

    HTML5

    NodeJS


Features

ripplePay's simplistic design gives users a host of features including:

Send currency to any account linked to the Ripple XRP ledger

Request currency to be sent directly into your account

Search for users

Multi-level verification system to secure users information

Deposit currency to a ripplePay account

Access a display containing a users transaction history
